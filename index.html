A) Introduction

Apartment Management System for wordpress plugin is ideal way to manage complete housing society or neighbourhood maintenance tasks. It has different user roles like Admin, Resident members, Gatekeeper and Accountant Users.

This user guide covers all options that this plugin provides. Please read the documentation carefully as it probably contains answers to most of questions that may arise. If you feel some aspect is not covered or have some questions, contact us via e-mail sales@mojoomla.com. We will do our best to answer you in shortest time.

B) Installation:

1). FTP Upload

  1. Unzip the " apartment_management.zip " file
  2. Upload the extracted "Apartment Management" plugin folder into "/wp-content / plugins/" in your wordpress installation
  3. To Activate the plugin, Go to Plugins and activate the installed plugin

2). WordPress Upload

Steps for Apartment Management System plugin Installation:

  1. Go to Wordpress Admin Plugins → click on Add New → click on Upload.
  2. Unzip the "apartment_management.zip " file and Upload the " apartment_management.zip " . → click on Install Now.
  3. Click on Activate plugin.
  4. You can see Apartment Management on dashboard.

Increasing the WordPress Memory Limit
  • Edit your wp-config.php file and enter something like: 
    define('WP_MEMORY_LIMIT', '256M');
    WordPress memory can be different to the server â€“ you need to set this regardless of server memory settings
  • If you have access to your PHP.ini file, change the line in PHP.ini If your line shows 32M try 256M: 
    max_execution_time= 120 ; Maximum execution time of each script 
    memory_limit = 256M ; Maximum amount of memory a script may consume (256MB)
  • If you don't have access to PHP.ini try adding this to an .htaccess file:
    php_value memory_limit 256M
  • If none of the above works then talk to your host.

From this module you can set notification mail templates. You can also enable or disable this notification mail from general settings.



Each template also have shortcodes. So you can use that shortcodes in your mail template. Following Shortcodes are available in mail templates.

  • {{member_name}} : Display member name in mail.
  • {{unit_name}} : Display unit name in mail.
  • {{building_name}} : Display building name in mail.
  • {{loginlink}} : Display login link in mail.
  • {{rolename}} : Display role name in mail.
  • {{username}} : Display user name in mail.
  • {{password}} : Display password in mail.
  • {{notice_title}} : Display notice title name in mail.
  • {{notice_type}} : Display notice type in mail.
  • {{notice_valid_date}} : Display notice valid date in mail.
  • {{notice_content}} : Display notice content in mail.
  • {{event_title}} : Display event title in mail.
  • {{event_start_date}} : Display event start date in mail.
  • {{event_end_date}} : Display event end date in mail.
  • {{event_start_time}} : Display event start time in mail.
  • {{event_end_time}} : Display event end time in mail.
  • {{event_description}} : Display event description in mail.
  • {{nature}} : Display complaint nature in mail.
  • {{noticetype}} : Display notice type in mail.
  • {{noticecategory}} : Display notice category in mail.
  • {{complaintstatus}} : Display complaint status in mail.
  • {{description}} : Display complaint description in mail.
  • {{slotname}} : Display assigned slotname for parking in mail.
  • {{startdate}} : Display start date of assigned parking sloat in mail.
  • {{enddate}} : Display end date of assigned parking sloat in mail.

    I) Demo Theme Setup:

      1) FTP Upload

      Note: Existing customers please take a full backup of your site.
      1. Unzip the " twentytwenty-child.zip " file
      2. Upload the extracted "twentytwenty-child" theme folder into "/wp-content / themes/" in your wordpress installation.
      3. To Activate the theme, Go to WP admin panel Appearance > Themes activate the Twenty Twenty Child theme.

      2) Set Login and Registration page

        2.1) Set Login page
        1. To Set Login page go to the WP admin panel Pages.
        2. Edit the Apartment Management Login Page see the right side page setting on Template Dropdown select the apartment-login
        3. Click on Update button. (maybe shown Updating failed. The response is not a valid JSON response. but don't worry about it)



        2.2) Set Registration page
        1. To Set Registration page go to the WP admin panel Pages.
        2. Edit the Apartment Management Member Registration Page see the right side page setting on Template Dropdown select the apartment-registration
        3. Click on Update button.


      3) WP Menu Create (If not created)

      1. To Create WP Menu go to the WP admin panel Appearance > Menus
      2. Insert menu name and select the Desktop Horizontal Menu in Display Location and click on create menu.



      3. To add menu items Add menu items > pages
      4. Select Apartment Management Login Page and Apartment Management Member Registration Page and click on Add to Menu
      5. After menu items are added to menu then click on Save Menu button.


      4) Set Login page as a home page.

      1. To Set Login page as a home page go to the WP admin panel Settings > Reading
      2. In Your homepage displays select A static page
      3. In Homepage Select Apartment Management Login Page and click on Save Changes button.


      After Performing All Setup Successfully Visit your Site.

    K) Setting Up Language

    The steps to do the translation are as follows. For example let us consider French translation.
    1. Download and install Poedit.
    2. Open the file apartment-management / languages folder → apartment_mgt-fr_FR.po using Poedit .
    3. Choose the Source text and set its corresponding French text in Translation Text Area.
    4. Save the changes.
    5. In Admin menu >> Settings >> General >> Site Language >> Select Your Site Language.
